# Employee Salary Calculator

## Description

The Employee Salary Calculator is a simple web application developed using HTML, CSS, and JavaScript. It calculates an employee's salary based on the entered basic salary and displays the HRA, DA, PF, Gross Salary, and Net Salary in a table.

## Features

- Enter employee name
- Enter basic salary
- Automatic salary calculation
- Calculates HRA, DA, PF, Gross Salary, and Net Salary
- Displays salary details in a table
- Responsive and user-friendly interface
- Input validation for valid employee details

## Technologies Used

- HTML5
- CSS3
- JavaScript

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.

## Salary Calculation Formula

- **HRA = 20% of Basic Salary**
- **DA = 15% of Basic Salary**
- **PF = 12% of Basic Salary**
- **Gross Salary = Basic Salary + HRA + DA**
- **Net Salary = Gross Salary − PF**
