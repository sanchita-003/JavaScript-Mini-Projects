# Digital Stopwatch

## Description
This is a simple Digital Stopwatch created using HTML, CSS, and JavaScript. It allows users to start, stop, and reset the timer. The project demonstrates basic JavaScript concepts such as DOM manipulation, functions, variables, and the `setInterval()` method.

## Features
- Start the stopwatch
- Stop/Pause the stopwatch
- Reset the stopwatch
- Responsive user interface
- Displays minutes, seconds, and milliseconds

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
Digital-Stopwatch/

```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.
