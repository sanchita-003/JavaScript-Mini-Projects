# To-Do List

## Description

The To-Do List is a simple task management web application built using HTML, CSS, and JavaScript. It allows users to add, edit, complete, and delete tasks. Tasks are stored using the browser's Local Storage, so they remain available even after refreshing the page.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as Complete or Undo
- Delete tasks
- Stores tasks using Local Storage
- Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.

## How It Works

- Enter a task and click **Add Task**.
- Click **Complete** to mark a task as completed.
- Click **Undo** to restore a completed task.
- Click **Edit** to modify a task.
- Click **Delete** to remove a task.
- All tasks are automatically saved in the browser's Local Storage.
