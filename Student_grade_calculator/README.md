# Student Grade Calculator

## Description

The Student Grade Calculator is a simple web application built using HTML, CSS, and JavaScript. It allows users to enter a student's name and marks for five subjects, then calculates the total marks, percentage, and grade based on the entered data.

## Features

- Enter student name
- Enter marks for five subjects
- Input validation for marks (0–100)
- Calculates total marks
- Calculates percentage
- Displays grade based on percentage
- Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.

## Grade Criteria

| Percentage    | Grade|
|---------------|------|
| 90% and above | A    |
| 75% - 89%     | B    |
| 60% - 74%     | C    |
| 40% - 59%     | D    |
| Below 40%     | Fail |
