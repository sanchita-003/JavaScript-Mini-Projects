# Student Registration Form

## Description

The Student Registration Form is a simple web application developed using HTML, CSS, and JavaScript. It allows users to register by entering their personal details and validates the input before successful submission.

## Features

- Student registration form
- Input validation
- Email format validation
- Mobile number validation
- Password confirmation
- Gender selection
- City selection
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
Student-Registration-Form/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.

## Validation Rules

- All fields are required.
- Email must be in a valid format.
- Mobile number must contain exactly 10 digits.
- Password must be at least 8 characters long.
- Password and Confirm Password must match.
- Gender must be selected.
- A city must be selected.
